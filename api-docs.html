API Documentation¶

pyqtlet was designed to mimic the Official Leaflet API as much as possible, similar to how PyQt5 mimics Qt. This allows a large amount of laziness when it actually comes to writing the API docs for the module.

The documentation will only cover what methods have been implemented in pyqtlet. The details about functionality, options etc. should be obtained from the Leaflet site. In case of any discrepancies or specially implemented features, they will be mentioned here in the docs. Otherwise, expect the method to be identical to its JS counterpart.
