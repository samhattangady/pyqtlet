getJsResponse(js, callback)[source]¶

Runs javascript code in the mapWidget and triggers callback.

Can be used for custom use cases where information is required from the mapwidget, and the existing code does not cover the requirement

Parameters:
  • js (str) – The javascript code
  • callback (function) – The function that will consume the javascript response

Note

Qt runs runJavaScript function asynchronously. So if we want to get a response from leaflet, we need to force it to be sync In all that I have tried, I was unable to get the response from the same function, so I am converting it to a method with callback
